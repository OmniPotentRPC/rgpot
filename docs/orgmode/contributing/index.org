We welcome all contributions and contributors to ~rgpot~. Whether you
report bugs, improve documentation, or submit code, your help makes the tools
better for everyone.

* Communication and Conduct
We value a respectful and inclusive community. All contributors must follow our
[[https://rgpycrumbs.rgoswami.me/code_of_conduct.html][Code of Conduct]]. 

* Getting Started
Please review the technical requirements before submitting a Pull Request:

- Development Environment :: We use ~pixi~ for environment management. 
  Read the [[file:test_requirements.org][Testing & Environment Requirements]] to understand how to handle 
  conflicting dependencies.
- Documentation :: We author all documentation in ~orgmode~ converted to ~rst~,
  with the workflow and standards in the [[file:doc_builds.org][Documentation Standards & Authoring]] section.

* Reporting Issues
Use the [[https://github.com/HaoZeke/rgpycrumbs/issues][GitHub issue tracker]] to report bugs or suggest enhancements. Please
provide a minimal reproducible example for all bug reports. User facing comments
may be directed also to the [[https://matsci.org/c/eon/72][EON materials science section]].
