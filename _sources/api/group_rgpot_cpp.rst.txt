Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Safe C++ wrappers around the Rust-core C ABI.

The ``rgpot_cpp`` module provides move-only RAII handles and lightweight view/owning types that map to the C structs generated by cbindgen. Error handling is done via ``:ref:`rgpot::Error <doxid-classrgpot_1_1_error>``` exceptions, which are thrown whenever the underlying C API returns a non-success status code.
