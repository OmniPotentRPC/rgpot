Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Implementation of the RocksDB-based potential cache.

Header file for the :ref:`PotentialCache <doxid-classrgpot_1_1cache_1_1_potential_cache>` class.

This file implements the methods for the :ref:`PotentialCache <doxid-classrgpot_1_1cache_1_1_potential_cache>` class, handling the low-level interactions with the RocksDB library for persistent storage of potential evaluations.

This file defines the caching mechanism for the rgpot library, utilizing RocksDB to store and retrieve potential energy and force calculations.
